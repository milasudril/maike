Plug-in search paths

The plug-in search paths determines where Maike can find plug-ins. Whenever a plug-in is found in any of the mentioned directories, Maike will use that plug-in.

Search paths on a GNU system running on the Linux® kernel

On a GNU system running on the Linux® kernel, Maike looks for plug-ins in the following directories. The filename suffix .so is added automatically.

  1. The application load path
  2. /usr/local/lib/maike
  3. /usr/lib/maike
  4. ~/lib/maike
  5. Current working directory
